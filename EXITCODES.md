# Exitcodes
This is a collection of exit codes that are used in various programs. The exit codes are grouped into the following categories:
- Filesystem
- Subprocess
- Unknown

## Filesystem
| Exit Code | Description |
| --------- | ----------- |
| 1         | File not found |

## Subprocess
| Exit Code | Description |
| --------- | ----------- |
| 21        | Command not found |
| 22        | Permission denied |

## Unknown
| 101        | Unknown error |
